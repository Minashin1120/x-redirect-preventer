1. Data Collection & Usage

This extension (X Account Switch Redirect Preventer) does NOT collect, transmit, store, or share any personal information, passwords, browsing history, or other sensitive data with any external servers. All processing is executed locally within your browser.

2. Permissions & Purpose

  • storage: Used to locally save your agreement status to the terms.
  • webRequest: Used to detect the X account switch API call (switch.json) and temporarily hold the return URL.
  • webNavigation: Used to intercept the redirect to /home immediately after a switch and return you to the original page.
  • tabs: Used to facilitate the tab redirection to the previous URL.
  • declarativeNetRequest: Used for secure, high-priority redirection at the browser engine level.

3. Third-Party Sharing

Since the extension does not collect data, we do not provide, sell, or use any data for advertising or third-party purposes.

4. Security

Designed with security in mind, this extension avoids DOM manipulation and relies on standard browser APIs to minimize risks to your account.

1. データの収集と使用

本拡張機能は、ユーザーの個人情報、パスワード、閲覧履歴などを収集し、外部サーバーへ送信・保存することはありません。すべての処理はブラウザ内でローカルに実行されます。

2. 使用する権限

  • storage: 同意ステータスの保存に使用します。
  • webRequest: アカウント切り替えAPIの検知に使用します。
  • webNavigation: リダイレクトのインターセプトに使用します。
  • tabs: ページの差し戻し処理に使用します。
  • declarativeNetRequest: ブラウザエンジンレベルでの高速な遷移制御に使用します。

3. 第三者への提供

データを収集しないため、第三者への提供、販売、または広告目的での使用は一切行いません。

4. セキュリティ

DOM操作を行わない安全な設計により、アカウント凍結などのリスクを最小限に抑えています。
