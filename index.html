「Google APIから受け取った情報の使用は、限定使用要件（Limited Use requirements）を含むChrome Web Store User Data Policyに準拠します。」

1. データの収集と使用について

当拡張機能（X Account Switch Redirect Preventer）は、ユーザーの個人情報、パスワード、閲覧履歴、またはその他の機密情報を収集し、外部サーバーへ送信、保存、または共有することは一切ありません。すべての処理はユーザーのブラウザ内でローカルに実行されます。

2. 使用する権限とその目的

  • storage: ユーザーが規約に同意したかどうかのステータスをローカルに保存するために使用します。
  • webRequest: Xのアカウント切り替えAPI（switch.json）が呼び出されたことを検知し、その際のURLを一時的に保持するために使用します。
  • webNavigation: アカウント切り替え直後のリダイレクトをインターセプトし、元のページに戻すために使用します。
  • tabs: ページをリダイレクトから復帰させるために使用します。

3. データの第三者提供

当拡張機能はデータを収集しないため、第三者にデータを提供、販売、または広告目的で使用することはありません。

4. セキュリティ

当拡張機能は、DOM操作を行わずブラウザの標準APIのみを使用することで、アカウントの安全性を最大限に考慮して設計されています。
